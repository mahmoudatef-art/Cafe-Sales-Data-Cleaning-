#  Cafe Sales Data Cleaning

##  Project Overview

This project focuses on cleaning and preprocessing the Cafe Sales dataset to improve data quality and prepare it for Exploratory Data Analysis (EDA).

##  Objectives

- Data Cleaning
- Data Preprocessing
- Metadata Analysis
- Exploratory Data Analysis (EDA)

##  Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Missingno

##  Files

- Cafe_Sales_Raw.csv
- Cafe_Sales_Cleaned.csv

##  Data Cleaning

The preprocessing included:

- Removing duplicate records
- Handling missing values
- Standardizing categorical values
- Correcting data types
- Improving data consistency

> **Note:** UNKNOWN values were intentionally preserved because replacing or removing them could introduce bias into the dataset.

##  Exploratory Data Analysis(EDA)

EDA was performed using:

- Summary Statistics
- Metadata Analysis
- Distribution Analysis
- Bar Charts
- Line Charts
- Business Insights
